Week 01 · Foundations

Setup & Exploratory Data Analysis

Get a working environment, then learn to look at data before you model it.

Overview

This course is designed as a toolbox: every week adds a method, and the real skill comes from knowing which one to reach for and how to use it. In the first week, we will be setting up your Python environments and learn about basic data exploration. Manipulating data is the backbone of real ML, and a large portion of the work is in loading, cleaning, and handling data. We'll use the Titanic dataset as an example of a complete preprocessing pipeline, with techniques that can be reapplied anywhere.

By the end of this week you can…
  • Set up an isolated Python environment and install the course requirements.
  • Load a dataset with pandas and print basic statistics.
  • Spot and handle missing values, choosing a strategy per column.
  • Encode categoricals and scale features.
  • Summarise and visualise distributions, correlations, and outliers.

In motion

Exploratory Data Analysis. Profiling each column (dtype, nulls, stats), then distributions, then a correlation heatmap.

Data Cleaning. Four ways to handle missing values, one-hot encoding, and three scaling schemes side by side.

Descriptive Statistics. Mean/median/σ, a box plot built step by step, and what skew does to them.
