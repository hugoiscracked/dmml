Pen-and-paper, 2 hours, one A4 sheet (recto-verso) allowed. Scope: W8–W10: neural network fundamentals, training techniques, CNNs, RNNs/LSTMs, transformers and attention. Best of the two midterms counts.

Project clinic (Lecture 2)

You have a dataset, a baseline, and several methods tried. This session is dedicated to make sure that the project scope is airtight, regardless of the type of the data you chose to handle.

Final-push checklist
  • Baseline reported? Show how much each model beats a stupid baseline.
  • Breadth covered? Several model families, including at least one advanced method.
  • No leakage? Test set still untouched; preprocessing inside the CV pipeline.
  • Error analysis done? Where does the model fail, and why? (Don't just report a metric.)
  • Limitations written? An honest account of what doesn't work is a sign of rigor.
  • Reproducible? Seeds set, versions pinned, code runs.
